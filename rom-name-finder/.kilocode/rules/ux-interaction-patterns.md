# When implementing interactions

- Use React Hook Form for form handling
- Show validation errors inline with fields
- Use skeleton loaders for content loading
- Use spinners for action loading
- Implement toast notifications for success/error feedback
- Use modals for important actions (delete, confirm)
- Ensure all interactive elements are keyboard accessible
- Add proper ARIA attributes for screen readers
- Use consistent button styles and variants
- Implement proper focus management in modals
- Use React Router Link for internal navigation
- Add loading states to async operations
- Provide clear error messages
- Use consistent spacing and styling
