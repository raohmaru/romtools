# SQLite Search Web Application

A web application for searching in local SQLite databases using sql-wasm.js.

## Project Structure

- `frontend/` - React frontend application
- `frontend/public/db/` - SQLite database files
- `docs/` - Documentation

## Getting Started

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Features

- Search in SQLite databases
- Select different database files
- Real-time search results

## Technologies

- React + TypeScript + Vite
- SQLite with sql.js
- Vitest for testing
